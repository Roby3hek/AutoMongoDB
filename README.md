# AutoMongoDB
# 
Проект представляет собой десктопное приложение для управления автодилерским центром с использованием Python (Tkinter) и MongoDB в качестве базы данных. Программа реализует полноценную информационную систему для учёта автомобилей, клиентов и продаж с разграничением прав доступа (чтение, редактирование, администратор) и возможностью построения различных отчётов.
